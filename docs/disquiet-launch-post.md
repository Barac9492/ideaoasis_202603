## 제목

**IdeaOasis — 매일 아침, 실리콘밸리 스타트업 아이디어를 한국 시장 관점으로 분석해드립니다**

---

## 본문

안녕하세요, IdeaOasis를 만든 조여준입니다.

### 왜 만들었나요?

ProductHunt를 매일 확인하시는 분들 많으시죠. 저도 그중 한 명이었습니다. 그런데 매번 같은 질문이 남았습니다:

> "이거 한국에서도 될까?"

미국에서 뜨는 서비스를 보면서 한국 시장에 적용 가능한지, 이미 누가 하고 있는지, 규제는 어떤지 — 이런 맥락 분석을 매번 직접 해야 했습니다. IdeaBrowser 같은 영어 큐레이션 서비스는 있지만, **한국 시장 분석**까지 해주는 곳은 없었습니다.

그래서 직접 만들었습니다.

### IdeaOasis가 하는 일

**매일 아침 5시(KST)**, ProductHunt와 Reddit에서 가장 주목할 만한 스타트업 아이디어 10개를 AI가 큐레이션하고, 각각에 대해 한국 시장 관점의 분석을 제공합니다.

각 아이디어마다 포함되는 분석:

- **한국 시장 적합성** — 이 아이디어가 한국에서 통할 가능성
- **한국 경쟁사 현황** — 이미 비슷한 서비스를 하고 있는 한국 스타트업
- **규제 환경** — 한국에서 이 사업을 하려면 알아야 할 규제 사항
- **로컬라이제이션 전략** — 한국 시장에 맞게 어떻게 변형해야 하는지
- **네이버 검색 트렌드** — 실제 네이버 데이터랩 API 기반의 한국 소비자 관심도

### 기술 스택

- **데이터 소스**: ProductHunt API + Reddit (r/startups, r/SideProject)
- **AI 분석**: Claude (Anthropic) — 큐레이션 + 한국 시장 분석
- **트렌드 데이터**: 네이버 데이터랩 API (실제 검색량 기반)
- **프론트엔드**: Next.js + Tailwind CSS
- **인프라**: Vercel + Supabase
- **결제**: Stripe (프리미엄 구독)

### 무료 vs 프리미엄

- **무료**: 매일 상위 5개 아이디어 + 분석 전문
- **프리미엄 (₩9,900/월)**: 10개 전체 + 카테고리별 이메일 알림 + 주간 심층 분석

### 현재 상태

- 매일 자동 업데이트 운영 중
- 7일치 아카이브 제공
- 매주 토요일 주간 심층 분석 (TAM/SAM/SOM + 로컬라이제이션 플레이북)
- RSS 피드 지원
- 다크 모드 (토스/배민 인스파이어드 디자인)

### 앞으로의 계획

1. **Hacker News** 소스 추가 (Show HN)
2. **한국 경쟁사 자동 추적** — 글로벌 아이디어별 한국 스타트업 매칭
3. **타이밍 분석** — 미국 → 한국 시장 진입까지의 시차 분석
4. **개인화 다이제스트** — 관심 카테고리 기반 맞춤 추천
5. **커뮤니티 기능** — 아이디어별 댓글 + "나도 만들고 있어요" 선언
